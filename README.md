# Weasley - Working Prefect Server and Worker

## What is it?
It is an example script of Prefect self-hosted server and worker in one package. You just need to build the image, and used it in two separate compute instance in your favorite compute provider (hyperscaler or other PAAS).
Set the environment variable to determine one as worker and one as server. For server, you might need a database instance for data persistence.
Designed for low scale personal project.

## Why it called Weasley?
Because all four Weasley brothers are Gryffindor's prefect (Bill, Charlie, Percy, and Ron)
